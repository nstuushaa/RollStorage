# RollStorage API

Backend для работы со складом рулонов металла на C#.Net (Web API).

## Описание

Проект реализует RESTful API для управления рулонами металла на складе.  
Каждый рулон имеет следующие параметры:

- `Id` — уникальный идентификатор  
- `Length` — длина рулона (обязательно)  
- `Weight` — вес рулона (обязательно)  
- `AddedAt` — дата добавления на склад  
- `RemoveAt` — дата удаления со склада (необязательное)

API позволяет:

1. Добавлять новые рулоны на склад.  
2. Удалять рулоны с указанным `id`.  
3. Получать список рулонов с фильтрацией по диапазонам (id, длина, вес, дата добавления/удаления).  
4. Получать статистику по рулонам за указанный период.  

---

## Технологии

- C# .NET 7  
- ASP.NET Core Web API  
- SQLite (можно заменить на PostgreSQL при необходимости)  
- Entity Framework Core  
- xUnit + Moq для тестов  

---

## Установка и запуск

1. Клонировать репозиторий:
```bash
git clone <your-repo-url>
cd RollStorage
```
2. Настроить подключение к базе SQLite в appsettings.json:
```bash
{
  "ConnectionStrings": {
    "DefaultConnection": "Data Source=RollStorageDb.sqlite"
  }
}
```
3. Выполнить миграцию и создать базу:
```bash
dotnet ef database update
``` 
4. Создать проект
```bash
dotnet run
```
